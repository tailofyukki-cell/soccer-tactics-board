# サッカー戦術ボード（Soccer Tactics Board）

サッカーの戦術解説動画やフォーメーション説明に使える、ブラウザ上で動作する戦術ボードアプリです。
外部ライブラリを使用せず、HTML / CSS / JavaScript (Canvas) のみで構築されています。

## アプリ概要
- **対象環境**: PCブラウザ（ローカルファイルとして直接開いて使用可能）
- **機能**:
  - サッカーコートの2D描画（横長ピッチ）
  - 解説側チーム（白丸）の背番号・名前表示および編集
  - 相手チーム（濃緑丸）のポジション名表示
  - 各選手マークとボールのドラッグ移動
  - フォーメーションの初期配置（4-4-2, 4-2-3-1, 3-4-2-1, 4-3-3）
  - 表示切替（相手チーム、名前、背番号のON/OFF）
  - 盤面のローカルストレージ保存・読込
  - PNG画像としての書き出し機能

## 使い方

1. `index.html` をブラウザで開きます。
2. 画面上部の操作パネルから、フォーメーションを選択します。
3. 選手マークやボールをマウスでドラッグして、自由に配置を変更します。

### 選手名・背番号の編集方法
- 解説側チーム（白丸）の選手マークを**ダブルクリック**します。
- 編集パネルが表示されるので、新しい背番号と名前を入力します。
- 「OK」ボタンを押すか、Enterキーを押すと反映されます。

### マーカーの移動方法
- マウスでマーカー（選手、相手、ボール）をクリックしたままドラッグします。
- マーカーはコート内に収まるように移動範囲が制限されています。

### 保存・読込方法
- **保存**: 操作パネルの「保存」ボタンをクリックすると、現在の配置がブラウザのローカルストレージに保存されます。
- **読込**: 「読込」ボタンをクリックすると、最後に保存した配置が復元されます。

### PNG書き出し
- 「PNG出力」ボタンをクリックすると、現在の盤面（操作パネルを除く）が画像ファイルとしてダウンロードされます。動画のサムネイルや資料作成に活用できます。

### ローカル実行方法
- ダウンロードしたフォルダ内の `index.html` をダブルクリックして、ChromeやEdgeなどのブラウザで開くだけで動作します。
- Webサーバー（VSCodeのLive ServerやPythonのhttp.serverなど）でホスティングしても同様に動作します。
